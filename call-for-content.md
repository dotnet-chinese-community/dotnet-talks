# **📣 .NET Talks - 讲师招募**

## **活动介绍**

**_.NET Talks(dotnet-talks)_** 是一个致力于为 .NET 生态开发者提供分享、学习与交流的直播分享活动。我们希望通过 .NET Talks 把来自业界与开源社区的实战经验、最佳实践、新技术探索带给更广泛的 .NET 开发者／爱好者群体 -- 无论是初学者、中级开发者还是资深工程师。

## **我们在找谁**

如果你在以下任意一个方向有经验／话题，非常欢迎你来分享：

- .NET / C# 基础与进阶技巧
- .NET 运行时／框架／新特性
- .NET AI 实践与应用
- .NET 云原生开发监控
- 跨平台客户端开发（Windows / Linux / macOS / 移动）
- 性能优化、并发、异步 / 分布式微服务等场景
- 微服务 / 容器 / Kubernetes / DevOps / CI‑CD + .NET 相关实践
- 与数据库 / ORM / 数据访问 / 实践经验 (SQL Server, EF, NoSQL, etc.)
- 测试 / 自动化 / 代码质量 / 架构 / 设计模式 / 重构 / 维护经验
- 开源项目经验 / .NET 生态开源工具 / 实践分享
- 与 .NET 结合的新兴技术 ( WebAssembly, Cloud, IoT 等)
- … 以及你认为对 .NET 社区有价值的任意主题

不管你是独立开发者、公司工程师、开源贡献者、架构师、技术负责人，只要你觉得分享内容对大家有启发，都欢迎投递。

> 如果你有期待的讲师也欢迎推荐哈

## **我们期待你的加入！**

.NET Talks 的愿景是：让 .NET 世界更开放、更共享，也更贴近开发者真实的实践与需求。无论你是刚入门的小白，还是经验丰富的老鸟，只要你有热情、有干货、有故事 -- 我们都欢迎你来。
让我们一起，用技术和经验推动社区进步 🔥
