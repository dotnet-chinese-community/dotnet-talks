## 活动介绍

![yanshengjie](../../images/speakers/yanshengjie.jpg)

讲师：**颜圣杰**

微软最有价值专家，.NET 架构师，.NET Conf China 讲师，『.NET+AI』系列课程作者。

主题：**.NET Talks - .NET + AI 技术全景解密**

随着大模型能力不断演进，软件架构正从“功能驱动”走向“智能体驱动”。Function Calling、MCP、A2A 等协议正在重塑应用与模型的交互方式，使 AI 从“聊天工具”升级为真正可调度系统能力的核心引擎，圣洁为我们系统解读“.NET + AI”技术全景，聚焦 .NET 10 “AI Ready” 平台，串起从 Semantic Kernel、Microsoft.Extensions.AI 到 Microsoft Agent Framework 的演进路径，并结合简单示例，帮助大家快速理解核心概念，掌握在 .NET 体系下构建 AI 应用与智能体的关键思路。
