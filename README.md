# dotnet-talks

## Intro

dotnet talks 系列活动资料

> **_.NET Talks(dotnet-talks)_** 是一个致力于为 .NET 生态开发者提供分享、学习与交流的直播分享活动。我们希望通过 .NET Talks 把来自业界与开源社区的实战经验、最佳实践、新技术探索带给更广泛的 .NET 开发者／爱好者群体 -- 无论是初学者、中级开发者还是资深工程师。
> 我们在招募讲师，如果您有内容想分享欢迎报名，可以参考：[讲师招募](./call-for-content.md)

微软 Reactor 携手多位 .NET 技术专家开启全新系列专题 .NET Talks，为大家带来更多 .NET 技术直播分享。
